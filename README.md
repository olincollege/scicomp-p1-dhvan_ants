# Ants Project
Author: Dhvan Shah

## Files Setup

### main.py
Contains the Simulation class, responsible for high level logic. Will be the bridge between all the classes

### ant.py
Contains the Ant class, responsible for any logic related to a singular ant. This will contain the turning kernel and basic moving code

### gui.py
Contains the GUI class, responsible for displaying the ants and pheromone trails using pygame to update dynamically
